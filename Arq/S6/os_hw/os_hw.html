Windows vs Linux

Windows

  • Bootloader: Windows Boot Manager
  • Kernel: NT Kernel
  • Abstracción: HAL (Hardware Abstraction Layer)
  • Aplicaciones: Win32 / UWP

Utiliza Windows Boot Manager para iniciar y un kernel NT que gestiona procesos y abstracción mediante HAL.

Linux

  • Bootloader: GRUB
  • Kernel: Linux Kernel (Monolítico)
  • Servicios: Systemd o init
  • Módulos: Carga dinámica de drivers

Suele usar GRUB. El kernel Linux carga módulos dinámicos para manejar dispositivos y servicios del sistema.

Ambos sistemas arrancan el equipo, cargan un kernel, gestionan hardware, memoria y procesos, y ejecutan aplicaciones.
